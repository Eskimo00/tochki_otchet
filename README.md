# Отчет по точкам (GUI)

Небольшое настольное приложение на Python 3 + tkinter + openpyxl. Читает исходный Excel-отчет «Прохождение всех точек», агрегирует проходы по точкам 1–8 и формирует компактный Excel.

## Как работает
- Автоопределяет колонки «Название КТ» и «Продолжительность» по заголовкам, чтобы не зависеть от смещений столбцов.
- По строке `Модель: ... (N)` разбирает:
  - хозяйство (ТЭ, ТАН-1/2/3 и т.п.) — выводится отдельно и объединяется по группе;
  - название объекта без префикса хозяйства;
  - общее число точек на объекте (N в скобках).
- Сортировка: ТЭ, ТАН-1, ТАН-2, ТАН-3, затем остальные по имени хозяйства и объекта.
- Значения по точкам:
  - `---` — проходов не было;
  - `нет` — проходы были, но суммарное время 0;
  - `(количество) / чч:мм:сс` — есть проходы и время.
  Ячейки с `---`/`нет` красные, с данными зелёные, остальное — светлый фон; колонка «Хозяйство» объединена для подряд идущих моделей одной группы.

## Требования
- Python 3, tkinter (для GUI), openpyxl (чтение/запись Excel).
- Для сборки exe: PyInstaller.

## Запуск из исходников
```bash
python -m pip install -r requirements.txt
python report_gui.py
```

## Готовый exe
В каталоге `dist` лежит собранный файл `Отчет по точкам.exe`.
