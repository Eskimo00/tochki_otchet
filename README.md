# Точки-отчет (GUI)

Маленькая десктоп-утилита (Python 3 + tkinter + openpyxl), которая из исходного отчета Excel собирает итоговый файл:
- Читает блоки `Модель …` → строки с `Точка N` → завершение по `ИТОГО по ТС`.
- Дата берется из строки периода и ставится одинаковой.
- Итоговые колонки: `№ п/п`, `Название объекта`, `Дата`, `Точка 1…8` в формате `(кол-во) / чч:мм:сс`, при отсутствии — `нет`.
- Сортировка по названию объекта; автоподбор ширины колонок; заливки и границы уже настроены.
- При совпадении имени выходного файла добавляется суффикс `(1)`, `(2)`…

## Стек
- Python 3, tkinter (GUI), openpyxl (чтение/запись Excel).
- PyInstaller для сборки exe (готовый файл: `dist/tochki_otchet.exe`).

## Запуск exe (рекомендуется)
1. Запустите `dist/tochki_otchet.exe`.
2. Выберите исходный `.xlsx`.
3. Укажите место сохранения (или оставьте по умолчанию). Если файл есть, имя будет с суффиксом.
4. Нажмите «Сформировать» и дождитесь сообщения о готовности.

## Запуск из исходника (опционально)
```bash
python -m pip install openpyxl
python report_gui.py
```
